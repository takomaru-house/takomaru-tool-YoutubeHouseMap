# Sprint 1 — 管理者によるアニメーション・UI承認チェックリスト

Sprint 1 Exit Criteria：**🔴 管理者がアニメーション・UIを目視確認・承認する**ことが Sprint 2 進行の前提。

---

## 確認手順

ローカルでサイトを起動して各項目を目視確認する。

```bash
npm run serve   # http://localhost:8080 で公開サイトを起動
```

ブラウザで以下のURLを開く：
- **PC表示**：`http://localhost:8080/`（ウィンドウ幅 ≥ 768px）
- **モバイル表示**：開発者ツールでデバイスエミュレーション or ウィンドウ幅を 767px 以下に絞る

---

## ① 📱 モバイル（アコーディオン）

767px 以下の幅で確認。

- [ ] カテゴリ「施主目線」「専門家目線」の見出しが折りたたみ状態（▶アイコン）で表示される
- [ ] カテゴリをタップすると **300ms ease-in-out で展開** する（カクつかず滑らか）
- [ ] アイコン `▶` が `▼` に **90°回転アニメーション** する
- [ ] ジャンルをタップすると動画カード一覧が表示される
- [ ] 動画カードが **上から順にスライドイン（stagger 50ms）** する
- [ ] 再度タップで折りたたみアニメーションが逆再生される
- [ ] 動画件数が 0 のジャンルはアコーディオンから自動的に省略される

## ② 💻 PC（D3.js マインドマップ + サイドパネル）

768px 以上の幅で確認。

- [ ] ルート → 施主目線・専門家目線 → 各12ジャンルのツリーが描画される
- [ ] ジャンルノードにマウスホバーすると **円が拡大・色がアクセントカラーに変化** する
- [ ] ジャンルノードをクリックすると右サイドパネルに動画カード一覧が表示される
- [ ] サイドパネルの動画カードもスタガーアニメーション（50ms間隔）で表示される
- [ ] **マウスホイールでズーム** できる（最小0.3倍〜最大3.0倍）
- [ ] **マウスドラッグでパン** できる
- [ ] 「⟳ リセット」ボタンで初期表示に戻る（300ms アニメーション付き）
- [ ] サンプルデータが0件のジャンルは「このジャンルは現在収集中です。」と表示される

## ③ 🎴 動画カード（PC/モバイル共通）

- [ ] サムネイル画像が表示される（サンプルデータは架空IDのため **fallback `no-thumbnail.svg`** が表示されればOK）
- [ ] タイトルが2行を超えるとCSS で `...` 省略される（CAT-01 GNR-06 キッチンの最初のカードで確認）
- [ ] チャンネル名・投稿日（`YYYY年MM月DD日`）・動画時間（`MM:SS` または `HH:MM:SS`）が表示される
- [ ] **🔥 急上昇バッジ** が trending タグ付き動画に表示される
- [ ] **⭐ 管理者おすすめバッジ** が manual タグ付き動画に表示される
- [ ] バッジが付かない通常動画にバッジが表示されない
- [ ] カードクリックで **YouTube が別タブ** で開く（架空IDでも URL 形式は正しい）

## ④ 🎨 全体の見栄え

- [ ] フッターに免責事項（「本サイトはGoogleおよびYouTubeとは無関係...」）が表示される
- [ ] フッターに「YouTube利用規約」リンクがある
- [ ] ヘッダーに「最終更新日：2026年5月20日」が表示される
- [ ] PC↔モバイルでウィンドウサイズを変えると **768px 境界でレイアウトが切り替わる**
- [ ] ローディングスピナーが（一瞬でも）表示される動作確認

## ⑤ 🌐 クロスブラウザ（軽め確認、Sprint 5 で本格実施）

- [ ] Chrome で動作確認
- [ ] Safari（Mac/iOS）または Edge で動作確認
