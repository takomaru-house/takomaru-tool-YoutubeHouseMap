# テストケース生成

与えられた要求仕様・テスト計画・テスト観点一覧を分析し、ClaudeCodeで実行可能な具体的テストケースを生成します。

## カバレッジ戦略

```mermaid
flowchart TD
    A[261観点] --> B[優先度HIGH・自動化可能]
    A --> C[境界値・ビジネスルール]
    A --> D[セキュリティ・スキーマ]
    B --> E[約120ケース生成]
    C --> E
    D --> E
```
